TV Settings Import/Export

One-click bookmarklet to add Import and Export buttons to any TradingView strategy settings dialog.

Install

Drag this button to your bookmarks bar:

TV Settings IO

How to Install

Step 1: Make sure your bookmarks bar is visible (Ctrl+Shift+B)
Step 2: Drag the "TV Settings IO" button above onto your bookmarks bar
Step 3: Done! Go to any TradingView chart page

How to Use

1. Open a strategy settings dialog on TradingView (gear icon or double-click)
2. Click the "TV Settings IO" bookmark
3. Import (green) and Export (blue) buttons appear next to Cancel/OK
4. Click Export to download settings as JSON, or Import to load from a JSON file
Note: The buttons persist while the dialog is open. You need to click the bookmarklet once per page load (after refresh).

Alternative: Console Paste

If bookmarklets don't work, open F12 > Console and paste the contents of export-strategy-inputs.js.
